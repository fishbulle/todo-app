# Todo App

A simple todo list app built with React, TypeScript, and Vite.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [pnpm](https://pnpm.io/) — or npm if you prefer

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/fishbulle/todo-app.git
cd todo-app
```

### 2. Install dependencies

```bash
pnpm install
```

Or with npm:

```bash
npm install
```

### 3. Start the backend

The app expects a backend API running at `http://localhost:8080`. Start it before running the frontend.

### 4. Start the development server

```bash
pnpm dev
```

Or with npm:

```bash
npm run dev
```

The app will be available at `http://localhost:5173`.
